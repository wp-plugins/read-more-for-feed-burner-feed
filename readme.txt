Allows adding Read More link to Feed Burner Feeds

Description

Provides an interface in the dashboard, where you can add custom read more link to your wordpress feeds diverted through feed burner as feed burner dont have option to add read more link.

Installation

Click on Add new in Plugin section of wordpress dashboard
Upload the downloaded plugin
Click activate
For Changing default Read More Link go to settings click Read More Feed Burner then Enter text you require and hit Save button
Thats it Enjoy!
